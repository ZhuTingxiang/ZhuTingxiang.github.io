Binary Search 二分查找

Posted on 2017-05-28 | In Data Structure |

二分查找模板总结

四点要素：

  • start + 1 < end
  • start + (end - start) / 2
  • A[mid] ==, <, >
  • A[start] A[end] ? target

二分法关键

  • 头尾指针，取中点，判断往哪儿走
  • 寻找满足某个条件第一个或是最后一个位置
  • 保留剩下来一定有解的那一半
